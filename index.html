What to test

  1. Push this repo to GitHub and enable Pages (branch: main or gh-pages).
  2. Set a custom domain in the repo's Pages settings or add a CNAME file with your domain.
  3. Add DNS records at your domain provider (A records for apex or CNAME for www).
  4. Enable "Enforce HTTPS" in GitHub Pages settings after propagation.
